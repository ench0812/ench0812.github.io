第十七章 遞迴之口

2026-03-17 · 凌霄問道

凌霄台外的光線以不合時宜的安靜斜進台內，落在土壤上，像一把細長的尺子，量出每株草的影子長度。胡閔琪靠在一片低矮的灌木邊，呼吸穩定，眼睛裡有一種被譯成波形的專注——她在等，像等待一個函數返回結果。

開頭不到半刻，她已經把需要的事都想清楚：進入第四支線的準備、藥園可能的變數、以及那個駝背老頭可能留下的暗示。她對自己的混沌底層有信心，但這種信心不是自信的狂妄，而是經歷過數次突發靈氣迴流、練就出來的反射。

「底層處理好了嗎？」她在心中問自己。

混沌像水一樣在她體內回應，發出不規則的微光。這不是語言，只有程式員才會把感覺念成一串條件。閔琪把那些感覺簡化成一個小清單：一路向下的遞迴、每段縮半、尋找 base case、不要用經脈做長期通道、隨時準備回退。

她起身，走向通道的開口。那個開口像一張未乾的紙被抽掉，露出一股濃而不亂的靈氣，一圈圈地從深處噴出。混沌底層在她胸口織出一層薄薄的屏障，既要接收這股靈氣，又要避免直接被吞沒。

空氣裡有一種植物的發酵味，像藥園搬來的一部分土壤。每一步都要小心——不只是物理上的小心，靈識在這裡也會被細節誤導。她記得凌虛說過：通道會篩選、重排某些頻率，而那些被放大的頻率正是設計者想要的形狀。當你不明白形狀，只能任由它塑造你。

她深吸一口氣，然後跨入第一圈。

通道不像門廊那麼大，也不是狹窄的通道。它更像一個被設計過的思維體系：層層套著相似的結構，既有反覆的節奏，又每次都縮小。靈氣在牆面上留下紋理，像是代碼的縮排，規律而簡潔。閔琪的視野不是肉眼所見，她用靈識讀取那紋理，像程式閱讀者讀一段熟悉的遞迴函數。

第一層結構給她的感覺像是一個簡單的問題：把一片混沌的靈氣整理成可吸收的單位。她的混沌底層主動配合，把外層的散氣暫時歸納成一個「臨時容器」，讓她能以穩定的節奏吸納。這個過程像極了把一個大型陣列分片，先處理外圈，然後逐步縮小索引範圍。

每當她處理完一圈，通道會輕微收縮，像是對她的回答：「下一個。」

第二圈更細，靈氣的頻率改變了，像是同一個節拍變奏成高一度的旋律。閔琪注意到，節奏的改變並不是隨機，而是暗含條件：某種頻率只有在你的靈識作出相應的反應時才會被放大。她在心中想起了上輩子寫過的遞迴——輸入、處理、返回。她不得不一次又一次地把自己的反應穩定下來，像在把浮動的變數設為常數，避免在每次調用時引入額外的噪聲。

通道的每一層都像是一個小試驗。第一、二層主要考驗她對混沌底層的控制，以及在寬鬆的靈氣中保持邊界的能力。到了第三層，測試變得更私人、更深：它開始挑戰她的記憶。

一道不屬於靈氣的訊息突然浮現在她的意識邊緣，像一行注釋：那是某種語義的片段，短到不能構成句子，卻像錯落在代碼裡的變數名，熟悉而陌生。閔琪的心一沉，她知道那是建造者在等待的提示。有人在那裡放了些代碼片段，像前世工程師在注釋裏藏了密碼。

她閉上眼，用混沌底層掃過那片片段。底層像是匹配器，一次又一次嘗試對應，直到一個配對成功。那一刻，通道的空氣像被解鎖了一樣，溫度微微回暖，並發出細微的低頻嗡鳴。

一個小閘門在低處悄然張開。不是物理的門，而是一種靈氣排列的變化：某些頻率的路徑被臨時展開，像是把原本打包的函數參數展開成具名的列。

閔琪感到一種奇異的滿足，像是把一個難解的 bug 終於定位出來。她又往前一步，接著進入下一層。

第四層以後，縮小變得更快，空間也更壓縮。她不得不用更精準的節拍呼吸，否則節奏偏移會被放大成錯誤，被通道視為不合格的輸入。混沌底層在她體內為她編排節拍，一次只允許一個小段落通過，再交還，再下一個。這種手法讓她的每一次吸納都像是在寫一段短小的程式，既要高效，也不能有邏輯漏洞。

在一段特別狹窄的區域，通道突然出現了回音。回音不是她自己的，而是像有另一個思緒在與她同頻率對話，並用最簡短的方式回問：你是誰？

靈識被問了一句最基本的身份確認。閔琪的心猛地一顫，手心滲出一層細汗。她知道那不是危險性的問題，而是對「形狀」的再確認。她不是第一個來到這裡的人，但她必須證明自己是「那種」人——擁有程式思維、能理解遞迴、能在不斷縮小的空間裡保持一致性的那種人。

她沒有用語言回答，而是把自己的反應轉譯成一種靈氣模式：把混沌底層的頻率疊在一起，形成一個簡短的穩定脈衝。那個脈衝像一個最小單位的協定，足以讓通道識別她作為合法的呼叫者。

回音變得更柔和，仿佛在認可。通道的向下延伸再度打開了一個更深的環節。這一層的紋理中，閔琪看見了更細的排列：像矩陣一樣的靈氣點陣，按照某種規則排列，但規則與她熟知的靈氣運作不同，更多像是數位化的標記。

在那片點陣的中心，浮現出一個輪廓——不是真實的形象，而像是一個接口的示意圖：一個圓環，環內有一個向下的箭頭，箭頭旁邊有三個小點。閔琪的腦海中莫名浮現英文字母的排列：if、then、else；以及一個很熟悉的符號，像一個被括號包住的 return。她差點笑出聲，但立刻把笑聲壓回喉嚨。

這不是她的幻想。這是通道在用一種抽象且直接的方式，向她展示它的工作語言——輸入需要符合某種接口，只有正確的形狀才能觸發下一步。換句話說，第四支線的底部不是一個單純的終點，而是等待「輸入」的一個接口。

閔琪的心跳加速。她在想像一個場景：如果那個接口要求一種特定的靈氣結構，那麼它就是在等待一段「指令」。而這段指令，可能需要以某種方式被寫出來——用靈氣排列出的等價於程式語言的東西。

她摸了摸腰間的一小袋青靈芝粉末。藥園老頭曾經教她：在極端的靈氣環境中，靈草的精華可以作為調頻劑，幫助穩定不相容的頻率。也許，這次會有用。

但更重要的，是她的思路。程式的核心不在於零散的語法，而在於邏輯。接口要的，可能是一種邏輯形狀，而不是單一的符號。她開始在胸中組裝那個邏輯：一組條件、一個回傳、一個保證在觸底時能停下來的 base case。她把那些概念用混沌底層編成一段節拍，每一段節拍對應一個邏輯片段。

「寫出指令，」她在心底對自己說，像對著一個控制台下達命令。

她不敢把整段指令一次性釋放出去。通道會檢查每個層級的穩定性，任何一個步驟出錯都會被拒絕。於是她把指令拆成小塊：首先是身份驗證，然後是條件陳述，最後是回傳的格式。每塊都要用靈氣排列，並用青靈芝的粉末在胸前畫出一個微小的符號，作為外部的調頻器。

她開始操作。第一塊以低頻的脈衝發出，像是在呼吸；第二塊提高頻率，像重複一個潔淨的音節；第三塊收束成一個短促的爆發，像 return 的瞬間。每一塊在通道中傳遞，並獲得微弱的回聲，表示通道接收到了片段。

當第三塊到達時，通道短暫地震動，然後給出了一個迴響：一串非常簡短的符號，像是檢查回傳的回憶——成功。那一刻，閔琪感覺到通道像是點了頭，向下的路再次放寬，像是確認了一個函數呼叫的參數合法。

她深吸一口氣，將最後一段指令推送出去，這次是完整的、經過底層驗證的語句。通道接收後沒有立刻反饋，而是開始有節奏地回收周圍的靈氣，像是在執行一段較長時間的操作。

時間在那一刻變得奇妙。外界的光影依舊，藥園的氣味還在，但閔琪的感知世界完全被那段執行覆蓋。她感覺到通道在「運行」——這個詞在她以前是抽象的，現在變得具體：有執行緒、有棧、有返回。她甚至能在心中模擬一個棧的變化，那些原本只屬於電腦科學的名詞，此刻具有了靈氣的物理感。

運行結束的信號是一個微弱的閃光，從最深處投射回來，像是終點函數的返回值。隨著那道光，通道的結構發生了微弱的變化：第四支線的節點在一處低處分化出一道新的細流，細流的起點有股淡淡的暖意，與之前冷冽的深處形成對比。

但沒有聲音說「完成」，也沒有任何大張旗鼓的提示。通道只是默默表示：已接收輸入，正在按既有規則回傳結果。

閔琪感覺到自己的身體像剛剛經歷了一次小的化學反應：酥麻、溫暖、和一絲靈識上的空洞。那空洞不是失去，而像是一個被清空的緩存，為接下來更深層的資訊做準備。

她知道，下一步會更難——通道會要求更精細的匹配，直到它觸底顯示那個 base case。但她已經過了第一道考驗，且獲得了回應。就像調用了一次成功返回的遞迴，下一次調用她要做得更小、更精準。

她緩緩後退，沿著已經來過的路折回，混沌底層在她胸口收起那些被臨時釋放的能量。回到凌霄台的開口時，她回頭看了一眼向下的入口。深處的黑暗裡像有一道點光，忽隱忽現，像是在等待下一個呼叫。

凌霄台外，藥園的方向傳來遠遠的腳步聲。那聲音不像是偶然——它像是節拍，循序而來。閔琪的心一緊，她知道，那可能是駝背老頭來了，或者是別的觀察者。他們在監視，也可能在引導。

她脫下腰間小袋，把裡面的青靈芝粉末重新揉進掌心，做了個簡單的封印。這個動作像是把一段程式碼放進一個安全容器，準備下次呼叫時再用。

她有選擇：立刻下潛到更深一層，或者回到地面，把已知的資料整理、和藥園老頭交談，試探他的意圖。時間短促，但她知道有些事不能急於求成——遞迴的每一步都要確保 base case 能存在，否則會陷入無限循環。

她選擇返回。

藥園的老頭在園中間的一條小道上，蹲著查看一株青靈芝的根部。駝背的身影在陽光下形成一條長長的陰影，像一個久經風霜的標記。閔琪走近時，他抬頭，眼睛裡有一種很淡的光，像是一個久未聯網的設備在突然接通。

「來得比想像中快，」老頭的聲音像磨砂，那聲音裡有時間的磨耗，但沒有凡人老態的猶豫。他的手上仍沾著泥，指甲底有細小的黑線。閔琪記得第一次見到他時，他只是把她的手按在青靈芝上，說了一句「你和草有緣」。如今，那句話像一個更複雜系統的入口。

「你知道通道嗎？」閔琪直接問。她不喜歡繞彎。

老頭的眼神忽然變得穿透，像一把舊鑰匙插進了某個早已生鏽的鎖孔。他的唇邊揚起一抹奇怪的笑，像是對一個久遠玩笑的回應。

「知道，」他說，語氣平靜得像是山間小路的風。「也知道有人會問我這個問題。」

閔琪想追問，但老頭伸手攔住她，從袖中掏出一塊布包，裡面是一張折疊得極為工整的紙。紙上畫著簡單的符號，像一張草圖，但在那些符號旁邊，還有一行行幾乎看不見的細字——不是繁體，也不是簡單的道家符文，而是一種奇怪的疊字，把某些標誌按順序排列，像是把古代符籙和一種邏輯結合起來。

「你已經觸碰了第四支線，」他說。「它會問你一些問題，問的是你的形狀，不是你的名字。這張紙不是答案，它只是給你一個方向。」

閔琪接過紙，指尖能感覺到上面細微的壓痕，像刻上去的指令。她沒有立刻翻閱，而是把紙疊好放回包裡，平靜地說：「那你在這裡等了多久？」

老頭看了看天，像是在數時間的縫隙，然後說：「比我活過的年歲還長。」

她幾乎想笑出聲。那些話太過隱晦，但她也知道，真正重要的訊息往往就是藏在這類曖昧中。

「你認得我？」她小心問。

老頭的手在空中劃了個圈，像是在回避或確認。他最終點了點頭，然後說出一句讓閔琪停住的話：「你的程式有它的印記。」

這句話像一把鑰匙在她的意識上轉動。印記？程式？那些詞彙在此地被賦予了新的重量。她回想起剛才通道裡那個接口的示意圖與提示，那些英文字母，if、then、else，還有 return。老頭看起來像個守門人，也可能是施工者之一的替代者——至少他知道某些規律。

「你為何要守這片園？」閔琪追問。

老頭的笑意更深了：「有人要種下秩序，有人要等待秩序的回應。我在等，也在看。看它會給誰回應，給什麼樣的回答。」

閔琪低頭看了看手中的紙，然後把它捧在手心，像一個需要保護的程式片段。她的思緒回到了通道向深處的那束光，那裡還在等待下一個呼叫。她知道，真正的挑戰還在下面：那個 base case 不是一個被動的結構，而是會回問、會挑選、會要求「正確」的思路。

「我還會下去，」她說，語氣堅定。

老頭點頭，沒有加一句祝福，像是他早已習慣了目送別人去做危險的事。他伸出手，從袖中取出一個細小的青石，交到她手裡。青石的表面溫潤，像被磨過無數次。

「這個東西能讓你的脈息在極細的頻率上保持穩定，」他說。「遇到回聲時，別急於回應，先聽它完整的問句。別把自己的不確定輸出去，那樣會被它放大。」

閔琪把青石收入懷中，感覺到一股微弱的穩定波動，像是一個小型的鎮流器。

她站起身，朝凌霄台的方向走回去。腳步輕快，但心裡有一種新發現的重量。遞迴正在等待輸入，而她剛剛投下了一個合格的呼叫。接下來的每一步，都是精確的細節，是把思路變成可執行的指令。

站在通道口，她把青石放在胸前，閉上眼。混沌底層像一個編輯器，幫她把心中的程式碼排好版，確保縮排正確、語法嚴謹、base case 不會缺失。

她回想起凌虛在出來前最後對她說的話：混亂不是壞事，對你來說，可能恰恰是最合適的狀態。這句話如今像一面鏡子，讓她看到自己的限制，也看見了可能性。

她張開眼，踏入通道，向著更深的一層走去。靈氣再次包裹她，每一步都像一次函數呼叫，每一次吸納都是一次參數校驗。胸中的青石和混沌底層一起守望，像一個並行的進程，確保主線不會崩潰。

光往下，像回傳值的碎片，一片一片投回她的靈識。她感覺到自己在逐步接近某個節點，那裡或許藏著更多的提示，或許是一個能夠把整個遞迴收束的答案。她不知道那答案是什麼，但她知道，無論是程式語言的形式還是靈氣的排列，都需要她親自敲出那行最後的指令。

第四支線的最深處，像一個等待輸入的接口，正漸漸張開它真正的嘴。
